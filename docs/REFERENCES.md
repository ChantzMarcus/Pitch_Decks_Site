## üñºÔ∏è Image Optimization

### Tools
1. **Squoosh** - https://squoosh.app
   - Browser-based image compression
   - Compare formats (WebP, AVIF, etc.)
   - Free, privacy-focused

2. **TinyPNG** - https://tinypng.com
   - Batch compression
   - API available
   - Smart PNG/JPEG optimization

3. **ImageOptim** - https://imageoptim.com (Mac)
   - Desktop app
   - Lossless compression
   - Batch processing

### Image CDNs
- **Vercel Image Optimization** (built-in with Next.js)
- **Cloudinary** - https://cloudinary.com
- **imgix** - https://imgix.com

### Best Practices
- **Format**: Use WebP/AVIF for web, fallback to JPEG
- **Size**: Resize to max display size (no 4K images for 800px cards)
- **Compression**: 80-85% quality is usually imperceptible
- **Lazy loading**: Use `loading="lazy"` or Intersection Observer
- **Responsive images**: Provide multiple sizes with `srcset`
