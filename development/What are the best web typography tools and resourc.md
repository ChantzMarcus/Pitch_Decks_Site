## **Font Pairing Generators**

### **AI-Powered Tools**

**Fontjoy**[^1]

- Uses deep learning to generate font pairings
- Interactive slider controls contrast levels
- Lock individual fonts to explore combinations
- Editable preview text for testing with your own copy
- Free to use
- URL: fontjoy.com

**Monotype Font Pairing Generator**[^2]

- AI-powered with access to 1,000+ premium fonts
- Context-based style recommendations
- Expert-approved pairings
- Best for professional, curated results
- Paid subscription required

**Simplified AI Font Generator**[^2]

- Deep learning analysis for contrast levels
- High contrast for bold looks or balanced for subtle pairings
- Real-time copy preview
- Great for beginners


### **Curated Pairing Collections**

**Fontpair**[^3][^2]

- Specifically designed for Google Fonts matching
- Free Chrome browser extension for font identification
- Filter by serif, sans-serif, handwritten categories
- Database search with visual previews
- URL: fontpair.co

**Figma Google Fonts**[^4]

- Ready-made type pairing palettes
- Pre-configured in Figma
- Top 10+ proven combinations like Montserrat + Source Sans Pro, Lato + Merriweather
- Free within Figma

**Typotheque Font Combinator**[^5][^2]

- Curated font pairings rather than AI-generated
- Showcases classic combinations (Roboto + Merriweather, Open Sans + Lora)
- Excellent for inspiration and reference
- Supports 132 languages

## **Typography Scale Calculators**

These tools ensure harmonious visual hierarchy for your film showcase site.

**Elementor Typography Scale Calculator**[^6]

- Generates complete typographic scale from base font size
- Built on mathematical ratios (Golden Ratio, Major/Minor Thirds, etc.)
- Outputs pixel values directly for headings and body text
- Free, web-based
- URL: elementor.com/tools/typography-scale-calculator/

**Typescale**[^7]

- Create stunning typographic scales
- Generates CSS code automatically
- Visual preview of all heading levels
- Multiple scale ratio options
- URL: typescale.com

**Baseline Type Scale Generator**[^8]

- Free, easy-to-use tool
- Outputs CSS and Tailwind configurations
- Base font size input (typical: 16px)
- Preview text customization for realistic testing
- URL: baseline.is/tools/type-scale-generator/

**LGC Type Scale**[^9]

- Visual typography tool with CSS output
- Comprehensive spacing variables
- Unit converter (16px = 12pt)
- Full CSS template generation
- URL: layoutgridcalculator.com/type-scale/

**Fluid Type Scale Calculator**[^10]

- Generate font sizes with CSS `clamp()`
- Responsive design with viewport-dependent scaling
- Fluid scaling between minimum and maximum
- Perfect for modern responsive websites
- URL: fluid-type-scale.com

## **Variable Font Resources**

Variable fonts offer continuous style variations in a single file—ideal for premium film showcase design.

**Variablefonts.io**[^11]

- Comprehensive variable font resources
- Font libraries and implementation guides
- Educational content on variable font axes

**Google Fonts Variable Fonts**[^12]

- Free variable font library
- Five registered axes: Weight (wght), Width (wdth), Slant (slnt), Optical Size (opsz), Italics (ital)
- Full documentation on CSS values
- Over 200+ variable fonts available
- URL: fonts.google.com

**Wakamai Fondue**[^12]

- Inspect variable fonts to see available axes
- Visualize font capabilities before use
- Test font variations in real-time

**Samsa**[^11]

- Web app visualizing how variable fonts work
- Inspect glyph outlines and design data
- Explore designspace using sliders
- Educational tool for understanding font flexibility

**FontGoggles**[^11]

- Free, open-source macOS font viewer
- Preview available variable font axes
- OpenType features inspection
- Desktop application by Just van Rossum

**Adobe Fonts Variable Fonts**[^13]

- Interactive variable font panel in Photoshop, InDesign, Illustrator
- Customize font appearance with axis controls
- Slant, weight, and width adjustments
- Single variable font file contains all styles

## **Premium Film Showcase Font Selection Strategy**

### **Headings (Display Layer)**

- Founders Grotesk (premium grotesque, modern yet sophisticated)
- Archer (elegant, production-design quality)
- Montserrat Bold (Google Fonts alternative, strong presence)


### **Body/Descriptive Text**

- Merriweather (premium serif, readable, cinematic)
- Lora (elegant serif, film-friendly)
- Source Sans Pro (professional, clean)


### **Featured Text/Quotes**

- Beaufort or custom serif
- ITC Fenice (elegant serif from classic films)
- Epilogue Variable (contemporary, adjustable weights)


### **Technical Implementation**

- Use variable fonts to reduce file size (single file vs. multiple style files)
- Combine Google Fonts with Adobe Fonts for budget optimization
- Implement typography scale calculator for consistent hierarchy
- Use fluid typography for responsive scaling across devices
