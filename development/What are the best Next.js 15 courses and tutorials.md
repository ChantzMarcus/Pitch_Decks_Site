## üîë Key Topics Covered Across Resources

### App Router \& Server Components[^18]

- **Understanding**: Server Components render on server by default (less JS to browser)
- **Pattern**: Pass Client Components as children to Server Components
- **Lifecycle**: useEffect, browser APIs in Client Components only
- **Official Docs**: Next.js documentation with composing patterns


### Image Optimization[^19][^20][^21]

**Core Features**:

- Automatic format conversion (PNG/JPEG ‚Üí WebP/AVIF)
- Responsive images with srcSet \& sizes
- Lazy loading by default
- Blur placeholders (LQIP)
- CDN support (Vercel optimized)
- Cumulative Layout Shift (CLS) prevention

**Advanced**: Custom loaders, remote patterns, fill layouts, quality adjustments

### Animation Libraries

- **Motion.dev** (Framer Motion alternative)
- **Three.js + React Three Fiber** (3D animations)
- **GSAP** (Smooth transitions and complex sequences)
